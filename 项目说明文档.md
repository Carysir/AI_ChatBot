# 智能聊天机器人 — 项目说明文档

## 一、项目概述

本项目是一个基于 **RAG（检索增强生成）** 技术的智能聊天机器人系统，作为计算机本科毕业设计项目。系统支持自然语言对话、意图识别、知识库问答等功能，用户可上传自定义文档构建私有知识库，实现精准问答。

### 核心特性

- **多轮对话**：支持上下文连续对话，保持对话历史
- **意图识别**：自动识别用户意图（闲聊 / 知识问答 / 任务指令）
- **RAG 知识库问答**：上传文档 → 文本切分 → 向量化 → 语义检索 → LLM 生成回答
- **多模型支持**：兼容智谱 GLM、OpenAI GPT、通义千问等大模型
- **用户系统**：注册登录、JWT 认证、会话管理
- **知识库管理**：创建/删除知识库、上传文档、文档列表

## 四、RAG 知识库问答流程

### 文档导入阶段

```
用户上传文档(.txt/.md)
    ↓
文本切分 (按段落，chunk_size=500, overlap=50)
    ↓
Embedding 向量化 (text2vec-base-chinese 模型)
    ↓
存入 FAISS 向量数据库 (本地持久化)
```

### 查询阶段

```
用户提问
    ↓
意图识别 → 判断为"知识问答(qa)"
    ↓
Query Embedding → FAISS 向量检索 Top-K 相似片段
    ↓
构建 Prompt = 系统提示 + 检索到的文档片段 + 用户问题
    ↓
发送给 LLM → 生成回答（标注来源）
    ↓
返回回答 + 参考来源
```

## 六、数据库设计

### ER 关系

```
User (1) ──→ (N) Conversation (1) ──→ (N) Message
User (1) ──→ (N) KnowledgeBase (1) ──→ (N) Document
```

### 表结构

**users 用户表**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer, PK | 主键 |
| username | String(50), Unique | 用户名 |
| password_hash | String(128) | bcrypt 加密后的密码 |
| email | String(100), Unique | 邮箱（可选） |
| created_at | DateTime | 创建时间 |

**conversations 会话表**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer, PK | 主键 |
| user_id | Integer, FK → users.id | 所属用户 |
| title | String(200) | 会话标题（取首条消息前50字） |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

**messages 消息表**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer, PK | 主键 |
| conversation_id | Integer, FK → conversations.id | 所属会话 |
| role | String(20) | 角色：user / assistant / system |
| content | Text | 消息内容 |
| intent | String(50) | 意图标签：chat / qa / task |
| sources | Text | 知识库引用来源（JSON 格式） |
| created_at | DateTime | 创建时间 |

**knowledge_bases 知识库表**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer, PK | 主键 |
| name | String(100) | 知识库名称 |
| description | Text | 描述 |
| user_id | Integer, FK → users.id | 所属用户 |
| created_at | DateTime | 创建时间 |

**documents 文档表**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer, PK | 主键 |
| kb_id | Integer, FK → knowledge_bases.id | 所属知识库 |
| filename | String(255) | 文件名 |
| content | Text | 文档原始内容 |
| chunk_count | Integer | 切分后的片段数 |
| created_at | DateTime | 创建时间 |

## 八、环境配置说明

### `.env` 文件配置项

```bash
# ===== 大模型配置 =====
LLM_PROVIDER=zhipu                    # 大模型厂商：zhipu / openai / qwen
LLM_API_KEY=你的API Key               # 大模型 API 密钥
LLM_MODEL=glm-4-flash                 # 模型名称
# LLM_BASE_URL=                       # 可选，自定义 API 地址

# ===== 数据库 =====
DATABASE_URL=sqlite:///./chatbot.db    # SQLite 数据库路径

# ===== JWT 认证 =====
SECRET_KEY=你的随机字符串               # Token 签名密钥（生产环境务必修改）

# ===== RAG 配置 =====
CHUNK_SIZE=500                         # 文本切分大小（字符数）
CHUNK_OVERLAP=50                       # 切分重叠大小
TOP_K=3                                # 检索返回的文档片段数量
VECTOR_DB_PATH=./vector_store          # 向量库存储路径

# ===== 其他 =====
DEBUG=True                             # 调试模式
UPLOAD_DIR=./uploads                   # 上传文件存储目录
```

### 支持的大模型

| 厂商 | LLM_PROVIDER | 默认模型 | API 申请地址 |
|------|-------------|---------|-------------|
| 智谱 AI | zhipu | glm-4-flash | https://open.bigmodel.cn |
| OpenAI | openai | gpt-3.5-turbo | https://platform.openai.com |
| 通义千问 | qwen | qwen-turbo | https://dashscope.console.aliyun.com |

**推荐使用智谱 GLM**，注册后有免费额度，`glm-4-flash` 模型免费调用。

### SECRET_KEY 生成方式

```bash
python3 -c "import secrets; print(secrets.token_hex(32))"
```

## 九、部署指南

### 9.1 macOS 本地开发

**前置条件**：已安装 uv、Node.js 18+

```bash
# 1. 克隆代码
git clone https://github.com/Carysir/AI_ChatBot.git
cd AI_ChatBot

# 2. 配置环境变量
cp .env.example .env

uv sync                    # Python 依赖
cd frontend && npm install && cd ..  # 前端依赖

# 4. 一键启动（前台运行，调试用）

./service.sh logs      # 查看日志
```

## 十一、使用说明

### 1. 注册登录

- 打开浏览器访问前端页面
- 点击「注册」，输入用户名和密码，邮箱可选
- 注册成功后自动登录跳转到对话页面

### 2. 普通对话

- 在对话框输入文字，机器人会自动识别意图并回复
- 系统自动识别三种意图：
  - **闲聊(chat)**：日常对话，如「你好」「今天天气怎么样」
  - **问答(qa)**：知识查询，如「什么是机器学习」
  - **任务(task)**：翻译、总结、写作等指令

### 3. 知识库问答

1. 点击侧边栏底部的「知识库」进入管理页面
2. 点击「新建知识库」，输入名称和描述
3. 选择知识库后，上传 `.txt` 或 `.md` 格式的文档
4. 回到对话页面，在输入框上方的下拉菜单中选择对应的知识库
5. 提问时系统会自动检索知识库中的相关内容辅助回答
6. 回复会标注信息来源

### 4. 会话管理

- 左侧边栏显示所有历史会话
- 点击「新对话」创建新会话
- 点击已有会话可切换查看历史消息
- 鼠标悬停会话，点击删除图标可删除会话
